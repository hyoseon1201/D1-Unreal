# 작업 로그: 2026-05-03

## 작업 내용

### 1. 게임 데이터 레퍼런스 문서 생성
- **파일:** `GameDataReference.md`
- **목적:** 웹서버(Spring Boot) 연동 및 개발자 참조를 위한 데이터 스키마/태그 체계 정리
- **내용:**
  - Gameplay Tags 전체 레지스트리 (분류별 테이블)
  - Attribute 시스템 (Vital/Primary/Secondary/Meta) 및 `UD1AttributeInfo` 구조
  - Ability/Skill 시스템 (클스 계층, `UD1AbilityInfo` 구조)
  - Character Class 시스템 (`ECharacterClass`, `UD1CharacterClassInfo`)
  - Level-up 시스템 (`UD1LevelupInfo`)
  - Inventory 시스템 (`FD1InventoryItem`, `UD1InventoryComponent` RPC 인터페이스)
  - Damage & Combat 구조
  - DataAsset 종합 목록
  - 부록: 웹서버-언리얼 데이터 흐름 및 DB 테이블 후보

## 비고
- `.uasset` (DataAsset)의 실제 수치 데이터는 바이너리이므로 읽지 못함. 문서는 C++ 소스코드 기반 **스키마 및 태그 문자열**만 정리함.
- 실제 아이템/스킬 수치, 레벨업 필요 경험치 등은 에디터에서 수동 확인 또는 CSV/JSON Export 필요.
